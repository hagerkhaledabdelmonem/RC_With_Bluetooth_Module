# RC_With_Bluetooth_Module
It's a car controlled by bluetooth rc controller application.
### Component:
- Arduino UNO
- 4 Dc motors
- L298N Motor Driver 
- Bluetooth module using Bluetooth RC Controller Application
- Battery
